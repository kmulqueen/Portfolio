Band Finder

Social Media Application

Social Media Platform for the Music Industry

You can check out the app here!

Features

Designed to connect like-minded individuals in the music industry, Band Finder allows users to browse other profiles in the community, follow other users, create posts & comments, and like posts & comments. Once a user has registered or logged-in, they will be redirected to their dashboard. From the dashboard, users can edit their profile, see their followers & profiles they follow, and see their feed. The feed is populated with posts from whichever profiles the user if following. If the user wants to see all posts from all users or create a new post, they can visit the "Posts" page. If they want to see all profiles within the community, the can visit the "Community" page. Each users profile can be visited by any other logged-in user. Their profile displays a bio, instruments they play, bands/projects they are a part of, albums they have made, and any posts they have created. It also displays any other social media links the profile may have. Users and profiles can be deleted at anytime by clicking the "Delete My Account" button on their dashboard. This will also remove any posts the user may have created.

Building the App

I utilized the MERN stack to build this application. Node.js and Express to build my own REST API, MongoDB for my database, and React for the front-end. I also used the npm package mongoose to interact with MongoDB from the back-end, and React-Redux to manage the application's state. I built the mobile-friendly theme using sass. You can check the code for the theme here. This application is hosted on Heroku.

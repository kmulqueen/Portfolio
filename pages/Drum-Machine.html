Drum Machine

Drum Sequencer Application

Create Unique Drum Grooves

You can check out the app here!

A simple layout allows for maximum creativity. Select the number of pads you want to work with, click on some pads to activate them, and then press the play button. From there you can edit the pads and create some seriously cool drum patterns in the moment. You can mute tracks, change sounds, adjust tempo, and even program patterns in odd time signatures!

Building the App

I utilized Object Oriented Programming principles and built the sequencer using JavaScript ES6 Classes. To achieve the visual emphasis when the sequencer is running I used CSS animations and dynamically assigned them in JavaScript.

LIRI Bot

Command Line Node Application

Language Interpretation and Recognition Interface

Using the command line, users are able to pull their recent tweets, get song information through spotify, or get movie information through IMDB. LIRI can also read from a random.text file full of random commands and searches such as spotify-this-song, "I Want it That Way".

Tech Used

  • Node.js
  • Spotify API
  • Twitter API
  • IMDB API
